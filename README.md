# Script Fun 🎯

> 一个收集各种有趣脚本的仓库，包含 Shell 脚本、批处理脚本、自动化脚本等

## 📖 简介

本仓库致力于收集和分享各种实用的脚本工具，涵盖日常开发、系统管理、自动化任务等多个场景。所有脚本均经过测试，可直接使用或根据需求进行修改。

## 📁 目录结构

```
script_fun/
├── linux/          # Linux Shell 脚本
├── tampermonkey/   # Tampermonkey 用户脚本
└── README.md       # 本文件
```

说明：每个脚本一个目录，其中包含脚本文件和使用说明文件。


## 🚀 脚本分类

### Linux Shell 脚本
- 系统管理脚本
- 文件处理脚本
- 网络工具脚本
- 其他实用脚本

### Tampermonkey 脚本
- 网页增强脚本
- 自动化操作脚本
- 其他浏览器扩展脚本

## 💡 使用说明

1. **克隆仓库**
   ```bash
   git clone https://github.com/fanzhongwei/script_fun.git
   cd script_fun
   ```

2. **查看脚本**
   进入对应的目录查看具体脚本文件

3. **执行脚本**
   - Shell 脚本：确保有执行权限 `chmod +x script.sh`，然后运行 `./script.sh`
   - Bat 脚本：直接双击运行或在命令行中执行
   - Tampermonkey 脚本：在浏览器中安装 Tampermonkey 扩展后导入脚本
